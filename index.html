MSTR mNAV近似值-時間圖

本程式皆使用免費 API：

1. CoinGecko API：抓取 MSTR 公司 BTC 持倉量。

2. Alpha Vantage API：抓取流通股數與股價時間序列。

3. Binance API：抓取 BTC 日收盤價。

指標計算

- mNAV 近似值 =（股價 × 流通股數）/（BTC 持倉量 × BTC 價格）× 1.41。

- 此算法和 Strategy.com 的 mNAV 數據相比，兩者趨勢相近，但數值約差 1.41 倍，故在尾部乘上 1.41 。

- 最新市值（Alpha Vantage 概覽）：44,339,761,000 美元。

備註

- 由於使用免費 API 無法順利取得官方正式 mNAV，只能各處蒐集相關資料計算相似值。

- 除了上述誤差，免費 API 還可能有更新延遲與額度上限等問題，也會導致數據不準確。

- 股價與 BTC 價格來源不同，可能產生小誤差。
