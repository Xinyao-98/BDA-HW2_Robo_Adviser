MSTR mNAV近似值-時間圖

本程式皆使用免費 API：

1. CoinGecko API：抓取 MSTR 公司 BTC 持倉量。

2. Alpha Vantage API：抓取流通股數與股價時間序列。

3. Binance API：抓取 BTC 日收盤價（失敗時改用 CoinGecko）。

指標計算

- mNAV 近似值 =（股價 × 流通股數）/（BTC 持倉量 × BTC 價格）× 1.41

- 此算法和 Strategy.com 提供的 mNAV 數據趨勢相似，但數值約相差 1.41 倍，故在算式尾部乘上 1.41

備註

- 由於免費 API 無法順利取得官方正式 mNAV，於是改以蒐集其他指標來估算，並非完全準確。

- 免費 API 可能有更新延遲、限流或欄位變動等問題，產生誤差。

- 另外，股價與 BTC 價格來源不同，也可能造成些微誤差。
